
# Sales Data Analysis Dashboard

## Tools Used
- Power BI
- Power Query
- DAX

## Project Overview
Developed an interactive Power BI dashboard to analyze sales performance, track key business metrics, and generate actionable insights for decision-making.

## Data Processing
- Loaded raw data into Power BI Desktop
- Performed data cleaning and transformation using Power Query
- Conducted data profiling to ensure data quality

## Data Modeling
- Built relationships using primary and foreign keys
- Designed a star schema data model for efficient querying
- Managed cardinality and relationships between tables

## Key Features
- Created KPIs for sales, quantity, and profit analysis
- Designed interactive dashboards with slicers and filters
- Implemented different types of filters for dynamic analysis
- Developed top and bottom 5 product analysis based on:
  - Sales
  - Quantity
  - Profit
- Built trend analysis for sales over time

## Advanced Features
- Implemented DAX calculations for business metrics
- Configured filter interactions between visuals
- Customized filter behavior for dimension tables
- Improved user experience with interactive dashboards

## Insights
- Identified top-performing and low-performing products
- Analyzed sales trends over time
- Provided insights to support business decision-making
